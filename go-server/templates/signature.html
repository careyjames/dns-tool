Email Signature

Dark-background email signature with persistent styling across mail clients.

{{icon "envelope" "me-2 text-brand-gold"}}Preview
{{icon "info-circle" "me-2 text-brand-cobalt"}}Installation — Apple Mail
  1. Open the raw signature in Safari (not Chrome — Safari preserves rich-text copy fidelity for Mail).
  2. Press ⌘A (Select All), then ⌘C (Copy).
  3. In Mail, go to Settings → Signatures. Create a new signature (or edit existing).
  4. Click in the signature edit area, press ⌘A then ⌘V (Paste).
  5. The dark background is embedded as bgcolor + inline background-color on every cell, so it persists when sent.
{{icon "flask" "me-2 text-brand-violet"}}Design Notes
  • Background persistence: Every <td> carries both CSS background-color:#111827 and the legacy bgcolor="#111827" attribute. This dual approach is necessary because Gmail honors only CSS, Outlook honors only bgcolor, and Apple Mail honors both.
  • Gold accent lines: 3px top rule (gradient gold) and 1px bottom rule create the embossed-letterhead framing. The vertical divider uses the same muted gold at 25% opacity.
  • Owl border: Circular crop with a 2px gold ring border and subtle outer glow, matching the footer treatment.
  • Typography: System font stack (-apple-system) ensures native rendering. Gold (#c8a878) for links, light gray (#d1d5db / #9ca3af) for text hierarchy.
